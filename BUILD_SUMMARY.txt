USAGE QUICK START
================================================================================

1. AUTOMATED SETUP (Recommended):
   $ bash setup.sh

2. MANUAL SETUP:
   $ python3 -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt
   $ python main.py init

3. RUN SCRAPER:
   $ python main.py scrape              # One-time run
   $ python main.py scrape --export     # Run and export

4. START SCHEDULER:
   $ python main.py scheduler           # Runs every 2 days

5. EXPORT DATA:
   $ python main.py export --format all # CSV, JSON, Parquet
   $ python main.py export --format ml  # ML dataset

6. VIEW RESULTS:
   $ python main.py stats               # Database statistics
   $ ls data/processed/                 # Exported files

7. DOCKER DEPLOYMENT:
   $ docker-compose up -d               # Start service
   $ docker logs -f estate-crawler      # View logs
